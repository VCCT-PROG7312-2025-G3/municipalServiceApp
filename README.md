Municipal Service Application ReadMe

How to compile the program

/n •	Open Visual Studio 2022
•	Click “Open project or solution”
•	Find the project’s folder on your computer and select the “.sln” file.

Build the project
•	Go to “Build”, located at the top of your window
•	Then go to “Build Solution”

Run the application
•	Go to “Debug”, located at the top of your window
•	Select one of the following options:
o	“Start Debugging”.
o	”Start without debugging”.
•	The application will open in your default browser using localhost.

Use the features
•	Report an issue using the “Report Issue” option
•	Browse local events and through the “Local events and announcements” option
•	View your submitted requests in the “Service Request Status”
